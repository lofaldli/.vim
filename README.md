## installing vundle for plugin management
  
        git clone https://github.com/gmarik/Vundle.vim bundle/Vundle.vim
        vi vimrc
        :PluginInstall
  
## compiling YouCompleteMe
  
        cd .vim/bundle/YouCompleteMe
        ./install.sh
